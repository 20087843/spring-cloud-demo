Spring Cloud 实例程序(Greenwich.SR3)
===

#### 项目说明


#### 项目结构
1. config-server 配置中心，集成了消息总线自动刷新 client 配置
    - spring cloud config server
    - rabitmq
    - eureka client
2. register-center 服务注册中心
    - eureka server
3. monitor-center 应用监控中心
    - spring boot admin
    - eureka client
